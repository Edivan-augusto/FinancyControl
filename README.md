# FinancyControl

Webapp em Flask para importar extratos bancários (PDF) no formato “EXTRATO DE CONTA CORRENTE”, visualizar créditos/débitos, categorizar favorecidos (persistente em SQLite) e exportar CSV.

## Como rodar

```bash
pip install -r requirements.txt
python app.py
```

Acesse `http://127.0.0.1:5000`.
